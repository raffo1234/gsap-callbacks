Login Animated with GSAP and AngularJs
========================================
---



Dependencies
-----------
---
* [Node.js](https://nodejs.org/)
* [Bower](http://bower.io/)
* [Gulp](http://gulpjs.com/)

Install
------------
---
```
git clone https://git.yachay.pe/devteam/login-admin.git
bower install
npm install
```
Use
-----
---
```
gulp install
gulp serve
```
